<h3 align="center">Build entire apps by directing a swarm of AI agents — by voice, in one cockpit.</h3>

<p align="center">
  Execore is a local-first desktop platform where you brainstorm, plan, and ship software by
  <b>commanding a fleet of AI coding agents</b> — not by typing every line yourself.
</p>

---

### What's inside

- 🧠 &nbsp;**Execution Planner** — speak or type an idea → a structured plan → a live fleet of terminal agents executing in parallel, on a real-time timeline.
- 🐝 &nbsp;**ExeSwarm** — an autonomous multi-agent swarm: a coordinator brain dispatches agents across isolated git worktrees, with inter-agent mail and cost tracking.
- ✨ &nbsp;**Sonia** — a voice-and-hotkey conductor that drives the entire UI through a typed action bus and supervises your agent panes: delivering prompts, answering menus, and drafting replies.
- 🎙️ &nbsp;**ExeVoice** — push-to-talk voice control with multi-provider speech-to-text and automatic fallback.
- 🗂️ &nbsp;**Task Board · Recall memory · Integrated Browser** — a Kanban cockpit, per-workspace memory, and an in-app browser, so everything you need lives in one window.
